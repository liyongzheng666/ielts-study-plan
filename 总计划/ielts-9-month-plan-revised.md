# 雅思 9 个月备考计划 · 修订版（目标 7.5）

> 适用：基础偏弱、听说为短板的中国程序员
> 时间预算：周一~周五 50 分钟/天（40 核心 + 10 读背），周六/周日 各 120 分钟
> 周总量 ≈ 8.2 小时，9 个月 ≈ 300–330 小时

---

## 〇、先读这段：关于 7.5 的诚实预期

7.5 是**很高**的目标。从「偏弱」（约 5.0–5.5）到 7.5 等于提 ~2.5 个 band，
通常需要 **500–700 小时**。你的预算是 ~320 小时。

所以请把话挑明：

- **大概率结果**：执行到位 → **6.5–7.0**。
- **7.5 是冲刺目标**，要拿到，必须满足两个前提之一：
  1. 诊断测出你的真实起点不算太低（≥5.5），且听/读天赋不差；
  2. 或者在关键阶段**把工作日核心提到 60 分钟**、周末加到 150 分钟。
- **如果 9 个月到不了 7.5，正常**。延到 10–12 个月，或先考一次拿 7.0 再战。

这不是泼冷水，是让你**不会因为「我明明很努力却没到 7.5」而崩溃放弃**。
带着「冲 7.5、保 7.0」的心态执行，你反而走得最稳。

## 一、核心策略：用阅读和听力把平均分抬上去

雅思总分 = 四项平均，四舍五入到 0.5。**这条规则决定了 7.5 的最优打法。**

各项「提分天花板」和「所需训练类型」不同：

| 模块 | 提到 8.0+ 的难度 | 主要靠 |
|---|---|---|
| 阅读 | **低**（技巧型，可练到 8–8.5） | 同义替换 + 定位 + 速度 |
| 听力 | **中**（你弱，但天花板高） | 精听 + 听写 + 预测 |
| 写作 | **高**（最难提，需反馈） | 结构 + 语法准确度 + 反馈 |
| 口语 | **高**（需开口 + 反馈） | 流利度 + 发音 + 反馈 |

**到 7.5 的现实算法**——把高天花板的阅读/听力顶上去，托住写作/口语：

```
L8.0  R8.5  W6.5  S6.5  → (8+8.5+6.5+6.5)/4 = 7.375 → 7.5 ✅
L8.5  R8.0  W7.0  S6.5  → 30/4 = 7.5 ✅
L7.0  R7.5  W6.0  S6.0  → 26.5/4 = 6.625 → 6.5 ❌（写口拖垮）
```

**两条铁律：**

1. **阅读 + 听力是你的「提分主引擎」**，要往 **8.0–8.5** 打。
2. **写作 + 口语是你的「保底项」**，必须守住 **6.5（最好 7.0）**，绝不能掉到 6.0。
   ——所以听/说虽弱，但**绝不能拖到后期才练**，从第 1 个月就每天碰。

> 这一版相比旧计划最大的改动：**听力从第 1 天起每天练；写作/口语接上反馈；
> 阅读听力主动冲 8+；全程有模考和可量化验收。**

## 二、四大原则

```
1. 弱项天天碰，强项往死里抬
2. 输入必有输出，输出必有反馈
3. 用真题（剑桥 16–19），不要野鸡材料
4. 每两周一次量化体检，按数据调整，不按感觉
```

## 三、诊断测试（放在第 2 月末 · 地基期之后）

> 调整说明：原计划放在第 1 周。但对**偏弱基础**的人，开局裸考一套真题
> 既打击信心、又测不准「可训练上限」（很多分是地基没打好丢的，不代表天花板）。
> 所以**先用 1–2 月把地基打牢，月末再诊断**，得到的分数更干净、更能指导后续分配。
> 你自评**写作起点不低**——很好，那地基期写作可以直接上整段/短文，不用从单句爬。

第 2 月末花一个周末做**一整套剑桥真题（建议 Cambridge 16 Test 1）**：

- 听力、阅读：严格限时、对答案、换算 band。
- 写作 Task 1 + Task 2：各计时写完，丢给 AI 按评分标准打分（见第六节）。
- 口语：自己按三部分录音 13 分钟，AI/老师评估。

记录四项起点分，写进**第十四节的体检表**。**这是你后面所有「提升了多少」的基准线。**

## 四、工作日模板（周一~周五，50 分钟）

每天固定 5 分钟用 Anki 复习旧词（间隔重复），其余按下表轮转。
**关键：听力周一到周五至少出现 3 次。**

| | 核心 35 分钟 | 10 分钟读背 |
|---|---|---|
| **周一** | 精读 1 段雅思文章 + 整理 8 词/同义替换 | 泛读文章剩余部分 |
| **周二** | **精听**一段 Section 原文：听写 → 对原文 → 标盲点 | 跟读（shadowing）同段 2 遍 |
| **周三** | 做周一文章的阅读题 5–8 道 + 错题归因 | 泛读 / 高频词搭配 |
| **周四** | **精听 + 跟读**新一段（换口音/语速） | 听写难句订正 |
| **周五** | 口语：答 1 个小题，录音 60 秒 → AI 纠错 | 背 1 个写作/口语句型 |

> 工作日不碰整篇写作（时间不够会写废）。写作放周末整块时间。

## 五、周末模板（周六/周日，各 120 分钟）

周末是**整块、上强度、接反馈**的主战场。

### 周六 · 读 + 写专场（120 min）

```
40 min  限时阅读 1 篇 Passage（20 分钟做 + 20 分钟精析每道错题）
50 min  写作：Task 2 一篇（前期写段落，中后期写整篇）
20 min  把作文丢给 AI 按四项打分 → 看修改建议 → 重写 1 段
10 min  整理本周「同义替换」和「写作好句」入库
```

### 周日 · 听 + 说专场（120 min）

```
40 min  听力：完整 1 套 Section（4 段）限时 → 精析错题 → 听写盲句
30 min  口语：Part 1 + Part 2 + Part 3 各练 → 录音
20 min  自评 + AI 评估录音（流利度/语法/词汇/发音）→ 改 2 个问题
20 min  跟读 / 影子练发音（连读、词重音、句重音）
10 min  本周复盘，填体检表
```

> 进入第 7 个月后，**每两周把周末两天合并成一次完整四项模考**（见第九节）。

## 六、反馈回路：自学也能提分的关键（重点）

你是程序员，AI 用得熟，这是你相对其他考生**最大的优势**。把它用满。

### 写作：每篇都让 AI 当考官

固定用这个提示词（存成模板，每次复用）：

```
你是资深雅思写作考官。请严格按官方四项评分标准给这篇 Task 2 打分：
- Task Response（任务回应）
- Coherence and Cohesion（连贯衔接）
- Lexical Resource（词汇）
- Grammatical Range and Accuracy（语法范围与准确度）
对每一项：给出 band 分 + 具体扣分原因 + 1 条最该改的点。
最后：列出全文所有语法错误并改正；给一个 7.5 范文版本，
并用注释标出它比我的版本好在哪。
作文如下：[贴作文]
```

要点：
- **盯住 GRA（语法准确度）**——这是中国考生卡 6.5 的头号原因。
- 别贪大词，**用准 > 用难**。AI 标出的「中式表达」要专门收集成清单。
- Task 1（学术图表/书信）**必须练**，占写作 1/3，题型独立。

### 口语：影子跟读 + 录音 + AI 评估

- **每周 ≥3 次 shadowing**（影子跟读）：放原文，慢半句跟读，模仿语调连读。
  这是中国考生提「发音」分（占口语 25%）性价比最高的方法。
- 录音后用类似提示词让 AI 按 **流利度/词汇/语法/发音** 四项评，
  并要求「给我一个 band 7.5 的回答示范 + 我可以直接背的高分句型」。
- 有条件每周找 1 次真人对练（语伴/外教/口语 App），AI 替代不了真实互动压力。

## 七、语法 & 发音 syllabus（贯穿全程，每周 1 个点）

旧计划「每天 1 个句子结构」太虚。按下面清单走，每周吃透 1 个，融进当周写作和口语。

**语法（第 1–8 周各 1 个）**：时态体系 → 主谓一致 → 冠词(a/the/零) →
定语从句 → 名词性从句 → 条件句(if 三类) → 被动与强调 → 复杂句嵌套与标点。

**发音（贯穿）**：词重音 → 句重音/弱读 → 连读(linking) →
难点音 /θ/ /ð/ /r/-/l/ → 词尾辅音不吞音 → 升降调表达态度。

## 八、分阶段计划（含可量化验收标准）

### 阶段一 · 第 1–2 月：地基期（词汇 + 语法 + 听力入门 + 开口）

目标：建词汇/语法地基；**听力从第一天进**；养成每天开口习惯。

每周：3 天精读+词汇 / 2 天精听+跟读 / 周六读写 / 周日听说。

**验收（第 8 周末）**：
```
□ 听力 Section 1–2 正确率 ≥ 70%
□ 词库 ≥ 500 词条（带搭配/同义/例句），同义替换 ≥ 80 组
□ 写作起点不低 → 目标：能写出结构完整的主体段，并尝试 250 词整篇
□ 语法清单 8 个点全过一遍
□ Anki 每日复习不断签
□ 月末做诊断模考，得到四项基准 band（见第三节）
```
> 第 1–2 月的逐日可执行方案见单独文件：[`ielts-month-1-2-detailed.md`](../月度计划/ielts-month-1-2-detailed.md)

### 阶段二 · 第 3–4 月：阅读听力题型期 + 写作进场

目标：吃透全部阅读题型；听力 Section 1–4 都练；Task 1+Task 2 起步。

重点阅读题型：T/F/NG、填空、段落信息匹配、Heading 匹配、选择、配对。
重点：**题干词 ↔ 原文同义替换**的识别（这是阅读上 8 的命门）。

**验收（第 16 周末）**：
```
□ 阅读单篇限时 20 分钟，正确率 ≥ 70%（~28/40）
□ 听力整套正确率 ≥ 65%（~26/40）
□ 能独立写完 Task 2 整篇（250 词，40 分钟内）+ Task 1（150 词）
□ AI 评写作 ≥ 6.0；中式表达清单 ≥ 30 条并在修正
```

### 阶段三 · 第 5–6 月：四项齐头 + 反馈强化期

目标：阅读/听力冲 7.5–8；写作口语接满反馈，守住 6.5。

每周：阅读听力各 1 次限时 + 精析；写作 1 篇全程反馈；口语 3 次录音+评估。

**验收（第 24 周末）**：
```
□ 阅读 ≥ 7.5（~33/40）  听力 ≥ 7.0（~30/40）
□ 写作 AI 评稳定 6.5；口语自评/AI 评 ≥ 6.0
□ 口语 Part 2 能流利说满 1.5–2 分钟不卡壳
□ 发音清单（连读/重音）明显改善
```

### 阶段四 · 第 7–8 月：限时整合 + 模考期

目标：从「学习态」转「考试态」；练耐力和时间管理；阅读听力顶到 8。

**每两周一次完整四项模考**（听+读+写连考，口语当天补录）。
非模考周：单项限时 + 错题深挖。

**验收（第 32 周末）**：
```
□ 模考 听力 ≥ 7.5  阅读 ≥ 8.0
□ 写作 ≥ 6.5（争 7.0）  口语 ≥ 6.5
□ 模考总分 ≥ 7.0，且无单项低于 6.5
□ 错题型已收敛到 2–3 类，有专项修复记录
```

### 阶段五 · 第 9 月：考前冲刺期

目标：稳定发挥；减少新内容；冲刺薄弱单项。

每周 1 次完整模考 + 弱项专练。最后两周：复盘错题、背熟高频同义替换、
固定写作结构、整理口语素材库、**停止输入新资料**。

**验收（考前）**：
```
□ 连续 2 次模考总分稳定 ≥ 7.0，目标项摸到 7.5
□ Task 1/Task 2 各有可复用结构模板
□ 口语 9 大话题各有 2–3 个素材，能即兴展开
```

## 九、模考节奏总览

```
第 2 月末    ：诊断模考 1 套（定基准，地基期之后）
第 3–6 月    ：每月 1 套（测趋势）
第 7–8 月    ：每 2 周 1 套完整四项（练耐力/时间管理）
第 9 月      ：每周 1 套（进考试态）
```

模考铁律：**严格限时、模拟真实顺序（听→读→写）、中途不停、一次性做完**。
单 section 练得再多，也练不出连续作战的耐力——这是很多人考场崩盘的原因。

## 十、单词 & 同义替换（沿用旧版 + 强化）

单词卡格式（保留，很好）：

```
word: impact
意思：影响          词性：noun / verb
搭配：have an impact on
同义：effect / influence
例句：Technology has a significant impact on education.
用途：写作/阅读高频
```

强化点：
- **多从精读文章里捞词**，靠泛读反复撞见来巩固，**别死背词表**。
- 用 **Anki 做间隔重复**，把「忘」交给算法管理。
- 同义替换是阅读/听力上 8 的核心，**每周新增 ≥ 10 组**，并在做题时主动验证。
- 9 个月目标词库 **1200–1800 高质量词条**（带搭配，不是孤立词）。

## 十一、各模块上 7.5 的技巧要点

**阅读（冲 8.0–8.5）**：先扫题干关键词 → 回原文找**同义替换**而非原词 →
T/F/NG 区分「矛盾(F)」vs「没提(NG)」是高频失分点 → 控时 20 分钟/篇，难题先标记跳过。

**听力（冲 7.5–8.0）**：精听必须**听写**（dictation）才有效 →
读题时预测答案词性/内容 → 抓 but/however/actually 等转折信号词 →
练不同口音（英澳加）→ 拼写和大小写错了照样扣分。

**写作 Task 2（守 6.5 争 7.0）**：审题先定**立场和论点**（Task Response 最重要）→
四段式：引言+两主体段+结论 → **语法准确度 > 词汇炫技** →
每段一个清晰主旨句 + 展开 + 例证。

**写作 Task 1（别忽略）**：学术图表抓**整体趋势 + 关键对比 + 极值**，不罗列所有数据；
书信看清三个 bullet 全覆盖。≥150 词，20 分钟。

**口语（守 6.5 争 7.0）**：流利度第一，**宁可说错也别长时间卡壳** →
Part 2 用「点 → 展开 → 举例 → 感受」撑满 1.5–2 分钟 →
答案具体化（给细节、给例子）远胜空泛 → 发音靠日常 shadowing 打底。

## 十五、风险与止损

```
风险1：7.5 到不了。 → 止损：先考一次锁定 7.0，再针对性二战弱项。
```
