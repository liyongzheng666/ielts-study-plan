## 使用方式

1. 先阅读总计划，明确阶段目标和时间投入。
2. 按当前月份执行对应的月度计划，并记录实际完成情况。
3. 参考学习模版整理每次精读、精听和输出成果。
4. 每月复盘一次，根据达标情况调整后续计划。
