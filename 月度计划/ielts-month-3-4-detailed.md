## 〇、本阶段定位与目标

地基期装好了发动机，这两个月开始**学开车的具体技术**——
雅思阅读/听力是**题型化**的，每类题有固定解法。把 6 大阅读题型逐一吃透，
听力 Section 1–4 全上，写作 Task 1 系统速成、Task 2 开始写整篇。

**第 16 周末验收：**
```
□ 阅读单篇限时 20 分钟，正确率 ≥ 70%（~28/40）
□ 听力整套限时正确率 ≥ 65%（~26/40）
□ 6 大阅读题型各有一张"攻略卡"且能说出解法
□ Task 1 六类图表都能写；Task 2 能独立写完整篇
□ 写作 AI 评 ≥ 6.0；"中式表达清单"≥ 30 条并在修正
□ 词库累计 ≥ 1000 词条
```

## 三、阅读 6 大题型攻略卡（贴墙上）

```
① True / False / Not Given
   顺序题，按题顺序定位。
   FALSE：原文有"明确相反"信息。  NOT GIVEN：原文没提，找不到相反。
   最大陷阱：用常识猜 NG → 错。规则：指不出矛盾句 = NG。

② 填空题（Completion）
   先看空格前后，预测词性 + 词数限制（NO MORE THAN TWO WORDS 要数词数！）。
   答案多为原文原词，但题干是同义改写 → 靠同义替换定位。

③ 段落信息匹配（Matching Information）
   乱序、可重复、最耗时 → 放最后做。
   抓题干里的"特征词"（数据/人名/例子），回原文扫这些标志。

④ 小标题匹配（Matching Headings）
   抓每段"主旨句"（常在首尾句），别被细节带偏。
   先排除明显干扰项；段落主旨 = 该段反复出现的核心概念。

⑤ 选择题（Multiple Choice）
   选项都是同义改写，逐项与原文比对，排除"偷换程度/范围"的干扰项
   （all/always/only 这类绝对词常是陷阱）。

⑥ 特征匹配 / 句子结尾配对
   先读选项库，回原文找对应；注意选项可能多于题目（有多余干扰）。
```

> 共同心法：**雅思阅读考的是"识别同义替换"，不是"读懂每个字"。**
> 每做完一组，回头标出"题干词 ↔ 原文词"的替换关系，攒进同义替换库。

## 五、Task 1 六类图表速成（第 9–12 周配套）

```
折线图  趋势词：rise / climb / decline / fall / fluctuate / level off / peak at
柱状图  对比词：A is higher than B / twice as many as / the largest
饼图    比例词：account for / make up / a quarter of / the majority
表格    抓最大最小 + 关键对比，别逐格念
流程图  全程被动 + 顺序词：firstly → then → after that → finally
地图    方位词：to the north of / be replaced by / be extended
通用结构：改写题目(1句) → Overview(2句抓总趋势/最值) → 主体段分组描述(2段)
字数 ≥150，时间 20 分钟。Overview 是 7+ 的关键，必写。
```
