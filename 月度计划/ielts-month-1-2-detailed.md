# 雅思第 1–2 月 · 地基期逐日可执行计划

> 总纲见 [`ielts-9-month-plan-revised.md`](../总计划/ielts-9-month-plan-revised.md)，本文件是它「阶段一」的落地版。

> 材料：只拆练 **C16 T1、C17 T1、C18 T1**；**C19 T1 封存到第 8 周诊断**。
> 诊断模考放在**第 8 周末**（本阶段结束时），不在开局。

## 〇、本阶段定位

地基期不是「刷题冲分」，是**把发动机装好**：词汇、语法、听力耳朵、开口习惯。
**所以本阶段以「精读 / 精听 / 跟读」为主，不追求满分限时**（限时训练在第 3 月起）。

**本阶段 4 件必须养成的习惯：**

```
1. 听力天天碰（你的弱项，且天花板高，第一天就上）
2. 每天开口录音（哪怕 1 句，对抗"哑巴英语"）
3. 生词进 Anki，靠算法管理"忘"
4. 每周吃透 1 个语法点 + 1 个发音点
```

**第 8 周末验收标准：**

```
□ 听力 Section 1–2 正确率 ≥ 70%
□ 词库 ≥ 500 词条（带搭配/同义/例句），同义替换 ≥ 80 组
□ 写作起点不低 → 能写结构完整的主体段，并能凑出 250 词整篇
□ 8 个语法点全过一遍，8 个发音点各练过
□ Anki 每日不断签
□ 完成诊断模考，拿到四项基准 band
```

## 四、笔记记录示例（完整一天长什么样）

以「第 1 周 周一：精读 C16 T1 Passage 1（假设是自然/环境类）」为例，
当天笔记应该包含下面几块。**照着这个格式记就对了。**

### 4.1 单词卡（每篇 6–8 张，进 Anki）
```
─────────────────────────────
word: vulnerable
意思：脆弱的；易受伤害的        词性：adj.
搭配：be vulnerable to (sth)
同义：fragile / at risk / susceptible
反义：resilient / robust
例句：Polar bears are highly vulnerable to the loss of sea ice.
来源：C16 T1 P1 ¶2
用途：阅读高频；写作环境/健康话题可复用
─────────────────────────────
word: decline
意思：下降；减少（n./v.）       词性：noun / verb
搭配：a sharp/steady decline in；decline rapidly
同义：decrease / drop / fall / reduction
反义：rise / increase / surge
例句：There has been a steady decline in the bear population.
来源：C16 T1 P1 ¶3
用途：阅读 + 写作 Task1 图表必备（描述下降趋势）
─────────────────────────────
word: conservation
意思：保护（自然/资源）         词性：noun
搭配：wildlife/nature conservation；conservation efforts
同义：protection / preservation
派生：conserve(v.) / conservationist(n.)
例句：Conservation efforts have slowed the species' decline.
来源：C16 T1 P1 ¶5
用途：环境话题写作/口语高频
─────────────────────────────
```

### 4.2 同义替换组（每篇 ≥3 组，这是阅读上分命门）
```
减少：decline = decrease = drop = fall = reduction
保护：protect = conserve = preserve = safeguard
因为(题干→原文常这样换皮)：because → due to / owing to / as a result of
"很多"：many → a large number of / numerous / a great deal of
```

### 4.3 听力听写盲点（周二/周四记，专治"听不出")
```
原句：The museum is open until half past five, except on Mondays.
我听成：...open to five... on Monday
盲点①：until 弱读成 /ʌntl/，被我吞掉
盲点②：half past five 时间表达没反应过来
盲点③：except 没听出 → 答案写反
对策：本周专练"时间/数字 + 连读"听写
```

### 4.4 好句 / 句型库（每天 1 句，写作口语直接复用）
```
功能：让步对比（写作/口语都能用）
While online tools offer flexibility, they cannot fully replace
the interaction of a real classroom.
→ 套用：While A has the advantage of X, it cannot replace Y.
```

### 4.5 写作段落 + AI 反馈（周六记）
```
【我写的主体段】
Online learning is convenient. Students can study at home. They
save much time. It is good for busy people.

【AI 按四项反馈】
- CC(连贯)：句子太短太碎，缺连接词，像列点不像段落
- LR(词汇)：convenient/good 重复且太基础
- TR(回应)：只有结论没有展开和例子
- GRA：句子结构单一，全是简单句 → 影响语法范围分

【改写到 6.5+（抄进句型库）】
One major advantage of online learning is its flexibility. Because
lessons are often recorded, students can study whenever it suits
them, which is particularly valuable for those juggling full-time
work. For instance, an employee can review a lecture late at night
rather than being tied to a fixed timetable.

【我的收获】
- 学到结构：主旨句 → 原因(Because...) → 具体例子(For instance)
- 收 1 个句型：which is particularly valuable for...
```

### 4.6 口语录音自评（周五/周日记）
```
题目：Do you like your hometown?（P1）
录音 58 秒，回放后自评：
- 流利度：找词卡了 2 次（"um..."）→ 练过渡词 well / actually / I mean
- 语法：说成 "I live there since I was young" → 应为 have lived（现在完成）
- 发音：three 读成 /sri/ → 本周练 /θ/
- 内容：太空泛 → 补一个具体细节（"...famous for its night market"）
【下次改 2 点】：① 现在完成时 ② 加 1 个具体细节
```

### 4.7 错题归因（周三/周日记，比对答案更重要）
```
题号：C16 T1 阅读 Q5（True/False/Not Given）
我的答案：FALSE     正确答案：NOT GIVEN
错因：原文根本没提这个对比，我自己脑补了 → 混淆"矛盾(F)"和"没说(NG)"
规则固化：FALSE 必须在原文找到"明确相反"的信息；
          找不到相反信息 = NOT GIVEN，绝不靠常识猜
下次动作：做 T/F/NG 时，先问自己"原文哪句和它矛盾？指不出 → NG"
```

### 4.8 每日打卡（5 个填空，30 秒搞定）
```
日期：第1周 周一
学习时长：50 min
材料：C16 T1 阅读 P1
```

## 六、本阶段最容易踩的 6 个坑

```
1. 只对答案不归因 → 错题归因比做新题重要 10 倍（第 4.7 节）
2. 背孤立单词不记搭配/同义 → 等于没背，留存率极低
3. 听力只"听个大意"不听写 → 精听必须落到笔头才有效
4. 怕开口、不录音 → 口语是练出来的，地基期就要每天出声
5. 写作不接 AI 反馈闷头写 → 自己看不见自己的语法错，会卡住
6. 想"刷快进度"跳精读精听 → 地基期偷的懒，第 7 月模考会加倍还
```
